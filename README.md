# Kernal64cart
A simple Kernal Cartridge for the Commodore C64, which has to get the /HIRAM signal for functioning 100%

# Status
This is a preliminary release. The cartridge Rev. 2 is fully tested and functional. The documentation is almost complete. The test is not jet documented, which is the reason for calling this a preliminary release.

Find the revision history in the Module Description.
